What is Fabric.css?

A brief introduction to the framework

A pure CSS3 framework based on Google Material Design

Fabric.css was built with usability and performance in mind. Being in early stages of development, Fabric still lacks most core components, as well as animations and responsive layout support. These features are planned to be added in Q1 2018.

Free and open source

The source code of Fabric.css is available on Codepen and Github free of charge, under Creative Commons Attribution 4.0 license.
